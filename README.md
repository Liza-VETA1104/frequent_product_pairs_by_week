# Анализ совместных покупок товаров

## Задача
Найти 2 товара, которые чаще всего покупаются вместе в одном чеке за каждую неделю.

## Решение
- Самоджойн таблицы покупок для генерации товарных пар
- Группировка по году, неделе и паре товаров
- Оконные функции для ранжирования пар внутри недели

## Оптимизации
- Фильтрация по минимальной частоте покупок
- Ограничение популярными товарами
- Возможность использования алгоритма Apriori для больших данных

## Технологии
SQL (BigQuery/PostgreSQL), анализ ассоциативных правил
